Automated Vulnerability Scanner

This project combines container security scanning and IaC static analysis into a single, user-friendly Python tool. It uses industry-standard open-source scanners: Trivy for scanning Docker container images and tfsec for scanning Terraform IaC code. The tool filters vulnerabilities by severity (High and Critical) and exports detailed reports with remediation suggestions and links to official security advisories.

  • Docker Image Scanner: Uses Python's subprocess module to invoke Trivy as a Docker container, parses JSON output, and filters for HIGH/CRITICAL vulnerabilities with remediation guidance.
  • Terraform IaC Scanner: Runs tfsec for static analysis, filters output by severity levels, and provides resolution sections and URLs for fixing issues.
  • Unified CLI: Interactive menu allowing users to scan Docker images, Terraform code, or both sequentially with flexible severity filtering options.
  • Report Generation: Exports filtered results to text reports for auditing and provides beautiful console output using the rich library.

Azure Honeypot with Sentinel Integration

My hands-on journey into cloud security where I built a Windows 10 VM honeypot on Microsoft Azure to understand real-world attack patterns. This project taught me how to deploy cloud infrastructure, configure enterprise SIEM tools, and analyze security logs to detect and investigate threats in real-time.

  • VM Provisioning & Network Security: Deployed Windows 10 VM in Azure portal, configured Network Security Groups to allow all inbound traffic, and disabled Windows Firewall to expose the honeypot and attract attack traffic.
  • Attack Simulation & Log Collection: Simulated brute-force login attempts by intentionally failing logins, then captured Event ID 4625 (failed logon) events in Windows Event Viewer to understand real-time security monitoring.
  • Sentinel Integration: Created Log Analytics Workspace, deployed Microsoft Sentinel, and configured "Windows Security Events via AMA" data connector to automatically forward security logs from the VM to the SIEM platform.
  • Geolocation Enrichment: Imported 54,000+ row IP geolocation database as a Sentinel Watchlist, wrote KQL queries using ipv4_lookup to enrich logs with geographic context for threat intelligence.
  • Attack Visualization: Built custom Sentinel Workbooks with advanced JSON configuration to create interactive world maps showing attack origins, enabling visual analysis of threat patterns and attack hotspots.
