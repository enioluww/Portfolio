Personal Cybersecurity Lab – Windows Server Blue Team Simulation

I built a personal cybersecurity lab simulating a Windows Server environment to develop hands-on skills in blue team operations and system administration. Using VirtualBox, I configured Active Directory, Group Policy, secure file sharing, DNS sinkholes, Sysmon logging, and simulated attack scenarios.

  • Active Directory Domain: Configured Domain Controller (Windows Server) and Domain Member (Windows 10) with static IPs in a Host-Only network.
  • Domain Joining & User Management: Joined client to domain, created security groups, and managed user accounts with RBAC.
  • File Server Security: Implemented NTFS permissions for departmental shares (least privilege).
  • Group Policy Hardening: Enforced policies (Control Panel/Command Prompt restrictions, account lockout, USB blocking, login banners).
  • Audit & Monitoring: Enabled login auditing and monitored Event Viewer logs for security events.
  • Brute Force Attack Simulation: Tested account lockout policies by triggering failed logins.
  • Sysmon Deployment: Set up Sysmon with advanced logging for deep system monitoring.

Volunteer Connect

Volunteer Connect is a full-stack web platform designed to bridge the gap between university organizations seeking volunteers and students eager to contribute their time and skills to worthy causes. The platform streamlines the process of posting, discovering, and applying for volunteer opportunities, making campus engagement more accessible and impactful for all participants.

  • How it works: Organizations can create accounts, post volunteer opportunities, and manage applications. Students can browse listings, apply for roles, and track their volunteer hours and experiences.
  • Technical Implementation: Builtsing React for the frontend, Node.js and Express for the backend API, MongoDB for data storage, and Docker for containerized deployment and development consistency.
  • Value: Volunteer Connect fosters a culture of service and skill-building on campus, helping students find meaningful opportunities and enabling organizations to efficiently recruit and manage volunteers. The platform’s intuitive interface and robust backend make it scalable and easy to maintain.
